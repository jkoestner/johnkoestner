## Overview

Morai is an actuarial tool that can analyze mortality interfacing with the SOA mortality tables as well as the mortality rates from the CDC.

## Features

- Includes a web interface for visualizing mortality data and trends.
- Provides tools for experience, forecast, and credibility analysis.
- Includes integrations from other actuarial resources such as CDC and SOA tables

## Background

The project started out to be a framework for creating a repository and training ground for building models to predict mortality. The idea expanded to be a comprehensive actuarial toolkit for analyzing mortality. This includes having a dashboard, predictive models, experience metrics, and more.

The project is still under active development as new features are continually being added.
